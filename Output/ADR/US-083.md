# ADR: Parametric Surface Curves Module (`src/visual/systems/parametricCurves.ts`)

**Status:** Implemented

**Context:** US-083 replaces the current helical band geometry in ribbonField with long coherent ribbons lying on parametric surfaces (helicoid, Möbius strip, torus knot). The existing ribbonField generated points along simple helices with random perpendicular spread, producing disconnected fragments rather than coherent sweeping ribbons.

**Decision:** Introduce a new module `src/visual/systems/parametricCurves.ts` exposing:
- A `sample(u, v, seed) → { position, normal, tangent }` interface per surface family
- Three families: helicoid, Möbius strip, torus knot (with seeded coprime (p,q) pairs)
- Session seed selects family and seeds parameters via existing PRNG infrastructure
- `selectCurveFamily(seed)` for deterministic family selection
- `getSampler(family)` to retrieve the sampling function

Additionally, added an `aCurveParam` float attribute (normalized u-parameter, 0–1 along ribbon length) to the `RIBBONFIELD_ATTRIBUTES` registry. This enables GPU-side length-aware deformation: bass drives coherent wave propagation along ribbons, treble drives per-point shimmer keyed to ribbon position.

New shader files `parametricRibbon.vert.glsl` and `parametricRibbon.frag.glsl` replace the `ribbonWarp` shaders for the ribbon system. The original `ribbonWarp` shaders remain available for other systems.

**Curve families:**

| Family | Self-intersection | Depth extent | Seeded params |
|--------|------------------|--------------|---------------|
| Helicoid | Bands cross at different heights | z = pitch × angle (rises with u) | R, pitch, wind count |
| Möbius | Inherent at twist closure | w·sin(v/2) | R, half-width |
| Torus knot | Knot crossings from (p,q) winding | r·sin(q·t), r ≥ 1.2 | R, r, coprime (p,q) pair |

**Consequences:**
- The `sample()` interface is reusable for future visual systems.
- `aCurveParam` is a required attribute in the ribbon system; the health gate validates it on first render.
- The `createRibbonField(config)` factory signature is unchanged; no breaking changes to mode registration, quality extraction, or compound modes.
- Torus knot (p,q) pairs are clamped to 12 known-good coprime values to avoid degenerate geometry.
- Möbius strip inherently provides self-intersection; helicoid and torus knot variants achieve depth overlap via seeded v-offset band placement.
- All 61 ribbon-specific tests pass including 36 new US-083 tests.
